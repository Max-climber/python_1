# 🧗‍♂️ Скалолазный гайдбук для Telegram

Полнофункциональный скалолазный гайдбук в формате Telegram мини-приложения, аналогичный Rakkup.

## 🚀 Функции

- 📍 Интерактивная карта скалолазных районов
- 📋 Детальная информация о маршрутах (сложность, высота, тип)
- 📸 Фотографии маршрутов и секторов
- 📊 Отслеживание восхождений и статистика
- ⭐ Добавление маршрутов в избранное
- 🔍 Поиск и фильтрация по сложности, типу, району
- 📱 Оффлайн-режим (кэширование данных)
- 🗺️ GPS навигация к началу маршрутов
- 👥 Социальные функции (комментарии, оценки)
